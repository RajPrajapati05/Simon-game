# Simon Game

A browser‑based version of the classic “Simon Says” memory game, built with HTML, CSS & JavaScript.

## 🚀 Features

- Sequence of colours/sounds is generated and shown to the player  
- Player must repeat the sequence — as it gets longer each round  
- If the player makes a mistake, the game ends and you can restart  
- Simple and clean UI for quick play  
- Responsive‑friendly design  

---

## 🛠 Technologies Used

- HTML5  
- CSS3  
- JavaScript (vanilla)  
- No backend required — runs entirely in the browser
